# Herokuへのデプロイ準備

* `heroku login`コマンドでログインして下さい
* `git config --global user.email "you@example.com"`
* `git config --global user.name "Your Name"``
* `git init`
* `heroku git:remote --app APPNAME`
* `git add .`
* `git commit -m 'first commit'`
* `git push heroku master -f`
